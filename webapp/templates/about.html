What is Multilingual FrameNet?

The project aims to create an integrated database connecting various linguistic resources (FrameNets) across languages, such as Berkeley FrameNet, DiCoEnviro, SweFN, and other framed semantic systems. We developed a database structure based on existing relations between frames and FrameNets, linking multilingual data. The project utilized information available on web pages for FrameNets (see links below). All collected, organized and marked up by us data is available on our GitHub and on this website.
